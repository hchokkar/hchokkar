# Himanshu Kumar Chokkar

This repository contains the personal portfolio and profile website for Himanshu Kumar Chokkar, an IT Security Manager and cybersecurity leader.

## What’s in this repository

- index.html — main single-page portfolio website
- age.jpeg — profile image used in the site
- Himanshu_Kumar_Chokkar_-_IT_Security_Manager.docx — resume / professional document

## Preview the site

You can open the site directly in a browser:

1. Open index.html in your preferred browser, or
2. Run a simple local server from this folder:

   python -m http.server 3000

   Then visit http://localhost:3000

## Purpose

This project showcases:

- the main portfolio homepage in index.html
- a dedicated services page in services.html

- cybersecurity leadership experience
- guest speaking and thought leadership
- expertise in IT security and risk management
- a polished personal brand presentation

## Notes

The site uses a modern neon/glassmorphism design and is tailored for a professional online presence. You can open the new services page directly at services.html.
